# yandex_vacancy_search
Ищите вакансии на яндекс работе в автоматическом режиме

## Потратил полтора часа на ручной поиск вакансии, аккуратно откликнулся расписывая диферамбы о себе — и получил отказ за 10 минут. Значит, с другой стороны уже стоит автоматизация. Шел 2026 год. Тогда кандидат тоже имеет право автоматизировать свою сторону: парсинг, фильтрацию и приоритизацию вакансий под свой стек? Подумал - сделал.

Запуск
1. скачать
2. виртуальное окружение
3. отредактировать .env по .env.example (p.s. нужен ключ опенроутер апи). Отредактировать profile.txt - просто написать о себе, это нужно для промта в llm
4. установить зависимости pip install -r requirements.txt
5. запустить python app.py
6. открыть в браузере http://127.0.0.1:5000
7. После нажатия "кнопки" запустить парсер или запустить обработку llm ЖДИТЕ т.к. парсинг вакасий и их обработка имеют определенный delay

Что появится после парсера:
```
Парсер завершен.
Ключи: ml, ds, llm, rag, data scientist, machine learning
Найдено/обновлено: 85
Удалено закрытых: 0
Удалено по blacklist: 1
```

7. Кнопка запустить обработку llm. Чанкует все вакансии и отправит на обработку опенроутер по апи

8. Кнопка сформировать отчет. Отчет упадет в папку \reports
